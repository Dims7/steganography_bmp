Steganography


The Steganography program works with text messages and .bmp files. It allows you to:
-Encode the message into a .bmp file
-Extract message from .bmp file


To run, you need to run the main.py file with arguments.
-e		Argument to say that you need to encode a message
-d		The argument is that you need to decrypt the message
-—Å		Argument to say that you need to delete the message (Compatible with -d)

-m		After this argument, you must enter a message (Compatible with -e)
-f		After this argument, you must enter the path to the file (Compatible with -e)
